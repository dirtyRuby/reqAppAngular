Description

A tool for capture and display HTTP requests.

This simple tool is useful during the development of apps that

integrate with external services, such as http clients, webhooks,

etc. The Requests Trap app ('Req App') provides these services with

an endpoint to which they can send requests and notifications.

Usage

Send request: '/traps/trap_name'

You can view all your traps at 'Traps'.

Click specific trap to view all requests or use '/traps/trap_name/requests'.

Click specific request to view full information on request.
